# PXL Pedals - ML x Web for AI Project - Team 5
This repository contains all of the work for both of Deadlines for the Machine Learning & Web for AI combined project assigment. The details of the assigment can be found in [`assigment.md`](assigment.md).

The group consists of 3 members all of Erasmus students for PXL Digital Spring semester 2025/2026.

## Repository overview
The repository has been split into two folders, to separete Machine Learning parts to Web for AI for ease of browsing. Each deadline project work has been put in own disticnt folder. 

### Deadline 1
The work for Deadline 1 consists of of two parts:
1. Data preprocessing, cleaning, analysis for Machine Learning
2. Gradio Exploratory Data Analysis App for Web for AI

The second part heavily depends on first one, as we need to process and cleanse the data. It is recommended to follow ``01-03`` steps for initial setup of repistory. Before starting the ``requirements.txt`` should be installed with ``pip install -r requirements.txt``
